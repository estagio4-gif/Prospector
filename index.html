Configurações

A chave é salva apenas no seu navegador (localStorage). Nunca sai do seu dispositivo. Obtenha em console.anthropic.com.

Habilita a seção de sanções (CEIS/CNEP/CEPIM) e contratos federais por CNPJ. Como a API da CGU exige token e não permite chamadas direto do navegador, aponte para o seu proxy serverless (veja proxy/transparencia-worker.js). O token fica no proxy — nunca no navegador.

Habilita a seção de processos e a marcação de teses "já ajuizadas". Requer contrato Jusbrasil (Consulta PRO) e o proxy serverless (veja proxy/processos-worker.js). O token fica no proxy.

Habilita decisores + LinkedIn, melhor telefone validado, faturamento e funcionários presumidos. Requer plano Econodata com API (token x-api-token no proxy — veja proxy/local-proxy.py). Consome 1 crédito por empresa.

Apaga os dados em cache (Receita, Transparência, Econodata…) para forçar uma consulta nova. Mantém suas chaves, proxies e métricas.

Métricas de uso (piloto)

Agregados anônimos deste navegador — nenhum CNPJ é armazenado. Use para reportar a tração do piloto.
